# Controle financeiro
<div></div>

## O app permite que usuários registrem gastos e investimentos, exibindo um mural e gráficos interativos para facilitar a visualização das finanças.

💻 Funcionalidades
- Registro de gastos e investimentos
- Exibição de gráficos dinâmicos com MPAndroidChart
- Armazenamento local com SharedPreferences (futuramente migrando para Room Database)
- Interface intuitiva e responsiva
- Adaptação do layout para diferentes tipos de transação

🛠️ 📱 Tecnologias utilizadas
- Kotlin + Android Jetpack
- MVVM Architecture
- LiveData & ViewModel
- RecyclerView + ListAdapter
- MPAndroidChart para gráficos
- SharedPreferences para persistência de dados (por enquanto)
